# Lab99

Landing page da Lab99, estúdio brasileiro de software especializado em Selfware: CRM personalizado, automações, agentes de IA e integrações.

## Como executar

Requer Node.js 20 ou superior.

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build de produção

```bash
npm run build
npm start
```

Os visuais da página são produzidos com CSS e SVG locais. O diretório `public/assets/lab99/` está reservado para futuros assets de imagem ou vídeo.
